GiftExample:本地图片上传,动态GIF表情图生成

二 打开本地图片功能

显示原理
1.1 利用<input>标签打开本地图片。
1.2 FileReader读取图片，生成base64字符串给Egret显示。
1.3 Egret中利用RenderTexture截取多张表情图的base64字符串，传给gifjs工具类生成动态GIF。
